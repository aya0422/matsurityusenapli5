# お祭り抽選アプリ

お祭りをテーマにした抽選アプリケーションです。

## 機能

- 時間帯に応じた当たり番号の設定
  - 前半（11:00-14:00）: 10分ごとに18個の当たり（1-18番）
  - 後半（14:00-17:00）: 20分ごとに9個の当たり（19-27番）
- 当たり時の演出
  - 紙吹雪エフェクト
  - 花火エフェクト
- ハズレ時の演出
  - 雨エフェクト
- お祭りをイメージした装飾
  - 提灯
  - 屋台
  - ガーランド

## 使用方法

1. リポジトリをクローン
```bash
git clone https://github.com/aya0422/matsurityusenapli.git
```

2. ブラウザで`index.html`を開く
